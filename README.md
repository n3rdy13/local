# Acoustic Rhythm Academy - Drum Learning Studio

An immersive, client-centered drum learning simulator built specifically to run locally on your laptop. All practice logs, streak stats, speed targets, and custom plans persist directly to your local sandbox database without relying on external servers or Firebase frameworks.

## ✨ Core Features
*   **Precision Web-Audio Metronome**: Highly accurate tempo scheduling, dynamic BPM sliders, accents, and multiple time-signatures.
*   **Interactive Rudiment Trainer**: Play along with diddles, flams, rolls, and drags. Score your tapping accuracy in real-time, matching left/right hand strokes.
*   **AI Coach Dave & Routine Synthesizer**: Connect with Gemini to generate smart workouts, receive tailored sticking tips, or request syncopated grooves.
*   **Groove Sequencer**: Design custom loops with Hihat, Snare, and Kick triggers, syncable to any custom BPM.
*   **Stats Tracker**: Monitor daily streaks, time allocation across rudiments, and precision trends with interactive Recharts.

## 🚀 Running the App Locally

This app resides in the [App Folder](file:///new_repository).

1.  Navigate into the directory:
    ```bash
    cd new_repository
    ```
2.  Install required dependencies:
    ```bash
    npm install
    ```
3.  Boot the application development server:
    ```bash
    npm run dev
    ```

> 💡 **Tip:** You can download the complete local environment snapshot (complete with source files and styling parameters) using the 'Download' button in the Environment settings of the AI Studio Playground!
